# 小绿书（微信图片帖）设计规范

## 平台特性

微信图片帖（小绿书）是在公众号内以图片序列展示的内容形式，最多 20 张图片。用户通过左右滑动浏览，完读率由每张图的视觉吸引力决定。

**与小红书的关键差异**：
- 受众：微信用户整体偏成熟，设计感接受度更高
- 浏览方式：横向滑动（vs 小红书的竖向滚动）
- 分享场景：朋友圈/微信群（更注重分享动机）

---

## 封面规范

- 比例：3:4 竖版
- 单一主题，文字 ≤25%
- 清晰传达内容价值主张
- 字体：可以比小红书更设计感，但避免过于沉闷的商务风

---

## 内容图规范

**密度**：
- 每张 ≤3 个核心信息点
- 文字占比 20-40%
- 留白 ≥30%

**视觉一致性**：
- 色调在整个图集中保持统一
- 字体风格保持一致
- 构图风格保持一致（勿混用极简和复杂风格）

**多样性**：
- 不同页背景/场景不重复
- 构图角度有变化

**禁止**：
- 与封面风格断裂
- 文字过密难以扫读
- 纯色大背景（除非整体为minimal极简风）

---

## Prompt 构建

封面：
```
{cover_hook} | {auxiliary_info}

风格：{style}

- 3:4 竖版构图
- 单一主题焦点，传达核心价值
- 文字≤25%，字体清晰可读
- 适合在微信信息流中吸引点击

```

内容图（批量）：
```
{paged_content_description}

风格延续：{style}
微信图片帖内容图，与封面风格一致，3:4 竖版

主题：{topic}
```
