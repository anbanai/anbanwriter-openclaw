# 内容合规规范

> 来源：微信公众号官方《侮辱谩骂案例解析》《低俗擦边案例解析》

## 概述

违反内容合规规范的文章将面临限流、删除或封号处理。创作时必须严格遵守以下规则。

## 语言合规

### 禁止暴力宣扬用语

- 禁止宣扬暴力行为的描述性语言
- 禁止美化或鼓励暴力冲突的表述
- 禁止将暴力行为描述为解决问题的手段

### 禁止粗俗用语（含变体）

以下形式均在禁止范围内：

- 直接使用粗口、脏话
- 谐音变体（用近似读音字代替）
- 省略变体（用"*"、"×"等符号替代部分字符）
- 字母缩写（英文首字母缩写替代中文脏话）
- 拼音缩写

### 禁止恶意攻击

- **谩骂**：对特定个人或群体使用侮辱性称谓
- **嘲讽**：以贬低方式描述他人的外貌、能力、身份
- **人身攻击**：针对个人特征（外貌、残障、民族等）的攻击性语言
- **威胁**：包含威胁语气或恐吓内容

### 合规替代建议

| 禁止表达 | 合规替代 |
|---------|---------|
| 侮辱性称谓 | 客观描述行为或观点 |
| 谩骂攻击 | 理性分析、论据支撑 |
| 情绪化威胁 | 陈述事实、法律途径 |
| 粗俗用语 | 标准书面语 |

## 内容合规

### 禁止低俗内容

- **敏感部位描述/聚焦**：对身体敏感部位的详细描写或特写
- **性暗示内容**：含有性暗示的文字描述、双关语
- **性行为描述**：直接或隐晦描述性行为过程
- **成人用品推广**：以任何形式推广或介绍成人用品
- **BDSM 及相关内容**：此类行为的描述、美化或推广

### 擦边内容警示

以下内容虽未明确违规，但应谨慎处理：

- 过度渲染的情感描写
- 以"健康科普"为名的不当内容
- 利用标题或封面图制造低俗联想

## 写作自检清单

发布前检查以下问题：

- [ ] 文章中是否存在任何形式的粗俗用语（含谐音、缩写）？
- [ ] 是否有针对特定人物或群体的侮辱性表达？
- [ ] 内容是否涉及性暗示或敏感描写？
- [ ] 标题和封面是否存在低俗联想的风险？
- [ ] 语气是否保持理性，无威胁或恐吓成分？

## 处理争议内容

当文章需要讨论敏感社会话题时：

1. **使用客观语言**：陈述事实，避免情绪化表述
2. **多角度呈现**：提供不同观点，不做单一倾向性判断
3. **引用权威来源**：以官方数据或专家观点支撑论述
4. **避免煽动性语言**：不使用可能激化读者情绪的措辞
